# Github-Final-Project 
Simple Interest Calculator

A simple Bash script that calculates simple interest based on the principal amount, annual interest rate, and time period in years. This project is part of the IBM Git & GitHub introduction course.

## Purpose
The purpose of this project is to demonstrate basic Bash scripting and Git version control workflows, including branching, merging, and pull requests.

## Usage
To run the script, ensure you have execute permissions and run it from your terminal:

1. Give execution permission:
   `chmod +x simple-interest.sh`

2. Run the script:
   `./simple-interest.sh`

## Input Fields
The script will prompt you for the following inputs:
* **Principal:** The initial amount of money (e.g., 1000).
* **Rate of Interest:** The annual interest rate as a percentage (e.g., 5 for 5%).
* **Time Period:** The duration in years (e.g., 2).

## Example
**Input:**
- Principal: 1000
- Rate: 5
- Time: 2

**Output:**
- The simple interest is: 100
